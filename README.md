# 技术栈

## 前端

1. Vue 3开发框架（提高页面开发的效率）
2. Vant UI（基于Vue的移动端组件库）（React版：Zent）
3. Vite（打包工具，特点：快！）
4. Nginx单机部署



## 后端

1. Java编程语言 + SpringBoot框架 
2. Spring + SpringMVC + MyBatis + MyBatis Plus
3. MySQL数据库
4. Redis缓存
5. Swagger + Knife4j接口文档
